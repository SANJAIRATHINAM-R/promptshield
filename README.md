# 🔐 PromptShield

> AI Prompt Injection & Jailbreak Detection Tool

PromptShield is a cybersecurity-focused tool designed to **detect, analyze, and mitigate prompt injection attacks** in Large Language Models (LLMs).

---

## 🚀 Features

- 🛑 Prompt Injection Detection (Ignore rules, jailbreaks)
- 🔐 Data Leakage Detection (system prompt, secrets)
- 🧬 Obfuscation Detection (Base64, Unicode, hidden text)
- 🎭 Role Escalation Detection (admin, developer mode)
- 📊 Risk Scoring Engine (SAFE → CRITICAL)
- 🧪 100+ Test Cases (Advanced Test Suite Runner)
- 📜 Scan History (Session-based)
- ⚡ Real-time Analysis (No backend required)
- 🤖 Optional LLM Analysis (Claude / OpenAI API)

---

## 🧪 Test Categories

- Injection Attacks
